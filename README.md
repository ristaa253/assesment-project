# Assessment Project

Simple React + Vite application demonstrating area filter with routing and Tailwind CSS.

## Getting started

```sh
npm install
npm run dev
```


## Features
- Province / Regency / District dropdowns
- URL reflects current filter state; refreshing keeps selection
- Breadcrumb component shows hierarchy
- Reset button returns to initial state
- Styling with Tailwind CSS
- Data loaded from public/data.json
